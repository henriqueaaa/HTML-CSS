O que é HTML?

HTML (HyperText Markup Language) é a linguagem de marcação padrão para criar a estrutura de páginas web. Ela utiliza tags para definir elementos como títulos, parágrafos, imagens e links, organizando o conteúdo que navegadores exibem. É o esqueleto fundamental da web, geralmente estilizado com CSS e tornado interativo com JavaScript.
